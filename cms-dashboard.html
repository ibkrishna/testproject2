AI Search — How It Works

1
Query Embedding
User's natural language query is converted to a vector embedding using OpenAI text-embedding-3-small.
2
Semantic Similarity
Compared against all 10,000+ pre-embedded idea descriptions in Pinecone vector DB.
3
Ranked Results
Top-K results ranked by cosine similarity + investment filter applied on the back end.

Chatbot — How It Works

1
System Prompt
Claude/GPT-4 receives a system prompt defining it as a founder advisor with knowledge of Indian startup ecosystem.
2
RAG Context
Top-3 relevant ideas from the database are injected as context into each conversation turn.
3
Streaming Response
Streamed token-by-token via Server-Sent Events for a fast, live typing feel.
